# week1-ai-assignment
**AIML Internship - Week 1 Project**

### Description
This project is part of the AIML Internship. It includes Python automation, API integration, and project documentation.

### Files Included
- `main.py`: Main Python script for the project.
- `api_task.py`: Script to fetch data from APIs.
- `tips.json`: JSON data file containing tips and quotes.
- `output.txt`: Output logs of the tasks.
- `report.pdf`: Detailed project report.
- `screenshots/`: Folder containing project output screenshots.

### How to Run
1. Clone the repository.
2. Run `main.py` using `python main.py`.
3. Run `api_task.py` to see the API integration results.
